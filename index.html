Esti pregatit pentru o lectie introductiva in HTML?

HTML -> este prescurtarea de la Hyper Text Mark-up Language si este codul care sta la baza paginilor web.
Structura de bază a documentului HTML este prezenta in imaginea de mai jos.

Elementul HTML

După declaraţia DOCTYPE, structura de bază a documentului HTML defineşte elementul HTML. Având în vedere că DOCTYPE nu se poate denumi drept un element adevărat, putem spune că elementul rădăcină al fiecărui document HTML este un element html. El începe cu tagul <'html>'de deschidere şi întotdeauna se termină cu tagul de închidere <'/html>.Câteva dintre cele mai importante caracteristici ale elementului html sunt următoarele:

  • reprezintă elementul rădăcină al documentului HTML;
  • este un element obligatoriu, fără el documentul HTML nu este valid;
  • cuprinde toate celelalte elemente de pe pagină.

Elementele head şi body se mai numesc şi antetul şi corpul documentului, respectiv acestea sunt părţi obligatorii ale fiecărui document HTML.

Titlurile (headings)

Fiecare document începe cu un titlu. Diferite părţi din document pot fi identificate prin subtitluri de diferite niveluri. Limbajul HTML cunoaşte 6 niveluri de titluri si este prezentat in imaginea de mai jos

Paragrafele (paragraphs)

În documentele HTML, paragrafele se creează prin introducerea textului în cadrul tagului corespunzător de deschidere <'p> şi de închidere <'/p>.

Ce sunt listele in HTML?

In HTML exista 3 tipuri de liste:

  • Ordonate (ordered) -> elementele se marchează cu unul dintre elementele de ordine, cel mai des cu cifre romane sau arabe, cu literele din abecedar etc.
  • Listele neordonate (unordered)-> spre deosebire de listele ordonate, la listele neordonate nu se pune accent pe importanţa ordinii elementelor, aşadar, elementele se marchează cu diferite simboluri, cum ar fi pătratul, cercul, liniuţa etc.
  • Listele de definiţie (definition) asigură crearea definiţiei fiecărui element în listă.

Tabelele in HTML

Fiecare tabel constă din celule aranjate în rânduri. Tagul de bază pentru crearea unui tabel HTML este tagul <'table>, în timp ce tagurile pentru crearea celulelor şi rândurilor sunt td, respectiv tr. Astfel, pentru crearea celulei se foloseşte tagul td - table data, iar pentru crearea rândului din tabel folosim tagul tr - table row.În exemplul de mai jos puteţi vedea cum arată toate acestea transformate într-un cod HTML:

Ce sunt linkurile?

Una dintre funcţionalităţile de bază ale limbajului HTML este posibilitatea de legare reciprocă a documentelor care se creează. Acest lucru se poate concluziona pe baza denumirii acestui limbaj: HyperText Markup Language. În lecţia introductivă a acestui curs am spus că noţiunea de Hypertext se referă la modul în care paginile HTML sunt reciproc legate prin linkuri care direcţionează la alte documente HTML. De aceea, putem trage concluzia că linkurile sunt mecanismul de bază pentru legarea mai multor documente HTML. Structura unui link este prezentat in imaginea de mai jos.
